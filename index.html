Введение

Folder Manager - это профессиональное программное обеспечение для управления папками и организации данных в проектах компьютерного зрения. Программа предназначена для автоматизации процессов переименования, проверки и организации папок с изображениями, полученными с устройств 10 и 12.

Основные возможности программы:

  • Автоматическое переименование папок согласно заданным шаблонам
  • Проверка целостности структуры данных
  • Управление настройками атак (диапазонами номеров)
  • Расчет времени съемки
  • Подробное логирование операций

Системные требования

Минимальные требования:

  • ОС: Windows 10/11, Linux с графической оболочкой, macOS
  • Процессор: 1 ГГц или выше
  • Память: 4 ГБ ОЗУ
  • Место на диске: 100 МБ свободного места
  • Python: 3.8 или выше (если запуск из исходного кода)

Рекомендуемые требования:

  • ОС: Windows 10/11
  • Процессор: 2 ГГц или выше
  • Память: 8 ГБ ОЗУ
  • Место на диске: 500 МБ свободного места
  • Разрешение экрана: 1366x768 или выше

Установка и запуск

1 Запуск исполняемого файла (рекомендуется)

Скачайте файл FolderManager.exe из папки с программой и дважды кликните по нему для запуска.

2 Запуск из исходного кода

Если у вас установлен Python 3.8 или выше, вы можете запустить программу из исходного кода:

# Перейдите в папку с программой
cd путь_к_папке_с_программой

# Запустите программу
python folder_manager.py

Важно: При первом запуске программа создаст файл конфигурации attack_config.json с настройками по умолчанию.

Обзор интерфейса

Программа имеет современный интерфейс с тремя основными вкладками:

1 Вкладка "🔄 Основные функции"

Левая панель содержит настройки операций, правая панель отображает логи выполнения.

2 Вкладка "🔍 Проверка"

Левая панель содержит элементы управления проверками, правая панель отображает логи проверок.

3 Вкладка "⚙️ Настройки атак"

Панель управления диапазонами номеров для различных атак и устройств.

Основные функции

Переименование папок

Автоматическое переименование папок из исходной директории в целевую согласно заданным настройкам.

1 Выбор папок

Нажмите кнопки "Обзор" для выбора исходной папки и папки назначения.

2 Настройка параметров

Выберите устройство ("Все", " 10" или " 12"), тип атаки и при необходимости отметьте опцию проверки содержимого.

3 Запуск процесса

Нажмите кнопку "🚀 Выполнить переименование" и следите за прогрессом в правой панели логов.

Пример структуры до и после переименования:

До: Исходная_папка/Папка_001/, Исходная_папка/Папка_002/, ...

После: Папка_назначения/02 2D Mask/91/, Папка_назначения/02 2D Mask/92/, ...

Функции проверки

Проверка атаки

Проверка целостности и правильности структуры отдельной атаки.

1 Выбор папки атаки

На вкладке "🔍 Проверка" в разделе "🎯 Проверка атаки" нажмите "Обзор" и выберите папку атаки.

2 Запуск проверки

Нажмите кнопку "🔍 Проверить атаку" для начала анализа.

3 Анализ результатов

Просмотрите результаты проверки в правой панели. Программа проверит количество папок, структуру содержимого и выявит ошибки.

Проверка ID

Комплексная проверка всех атак в папке ID.

Общая проверка проекта

Полная проверка всего проекта, включая все ID и атаки.

Управление настройками

Редактирование атак

Управление диапазонами номеров для различных типов атак и устройств.

1 Выбор атаки

На вкладке "⚙️ Настройки атак" выберите атаку из выпадающего списка.

2 Изменение диапазонов

Введите диапазоны для 10 и 12 в формате "начало-конец" (например, "91-170").

3 Сохранение изменений

Нажмите кнопку "💾 Сохранить" для применения изменений.

Вы также можете создавать новые атаки, переименовывать существующие и удалять ненужные атаки с помощью соответствующих кнопок.

Устранение неисправностей

Распространенные проблемы и решения

Проблема: "Не найден файл BestShot"

Решение: Убедитесь, что в каждой папке есть файл с "BestShot" в названии. Проверьте расширения файлов (.jpg, .jpeg, .png) и убедитесь, что файл не находится во вложенных папках.

Проблема: "Найдено X папок вместо 3"

Решение: Каждая папка должна содержать ровно 3 вложенные папки. Удалите лишние папки или добавьте недостающие.

Проблема: "Папка пустая"

Решение: Все вложенные папки должны содержать файлы. Проверьте каждую вложенную папку на наличие данных.

Проблема: "Недостаточно номеров в диапазоне"

Решение: Увеличьте диапазон номеров в настройках атаки или уменьшите количество исходных папок.

При возникновении других проблем: Сохраните логи (не очищайте их перед обращением в поддержку), опишите последовательность действий, приведших к ошибке, и укажите версию программы и характеристики системы.

Часто задаваемые вопросы

Ответы на самые распространенные вопросы о программе

Программа не запускается, что делать?

Убедитесь, что ваш компьютер соответствует системным требованиям. Попробуйте запустить программу от имени администратора. Проверьте, не блокирует ли антивирус запуск программы.

Как правильно подготовить исходные папки?

Исходные папки должны содержать только папки (не файлы) на верхнем уровне. Каждая папка должна содержать 3 вложенные папки и файл BestShot (изображение).

Что делать, если количество папок не соответствует ожидаемому?

Программа предупредит о несоответствии. Вы можете продолжить или отменить операцию. Проверьте исходные данные на целостность.

Как интерпретировать сообщения в логах?

✅ SUCCESS: Операция выполнена успешно
⚠️ WARNING: Предупреждение, не критично
❌ ERROR: Критическая ошибка, требует внимания
📋 INFO: Информационное сообщение
📄 DETAIL: Подробная информация

Можно ли отменить операцию?

Операции выполняются последовательно и не могут быть отменены во время выполнения. Рекомендуется делать резервные копии перед выполнением операций.

Как рассчитывается время съемки?

Время съемки рассчитывается на основе дат создания изображений. Учитываются разницы времени между первой и последней папкой каждого дня. Суммируется общее время съемки за все дни.
